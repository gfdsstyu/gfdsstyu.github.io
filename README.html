🔄

2-Way 학습: 플래시카드 모드

문제를 보고 답을 쓰는 '퀴즈 모드'와 답을 가리고 빠르게 넘겨보는 '플래시카드 모드'를 언제든 전환하며 입체적으로 학습할 수 있습니다.

🏆

학습 동기부여: 업적 시스템

'첫 100점 달성', '30일 연속 학습' 등 다양한 학습 목표를 달성하고 업적 배지를 모으는 게이밍 요소 도입으로 학습 동기를 부여합니다.

🚀 시작하기: 3분이면 충분합니다

Step 1. 웹사이트 접속

별도의 설치 없이 감린이 웹사이트에 접속하면 바로 시작할 수 있습니다.

Step 2. [필수] Gemini API 키 발급 (AI 채점/힌트용)

AI 채점/힌트 기능을 사용하려면 사용자 개인의 **Gemini API 키**가 반드시 필요합니다. 무료로 간단히 만들 수 있습니다.

  1. [**Google AI Studio**]에 접속하여 Google 계정으로 로그인합니다. (약관 동의)
  2. 좌측 메뉴에서 **Get API key (API 키 받기)**를 클릭합니다.
  3. 파란색 **"Create API key" (API 키 만들기)** 버튼을 클릭합니다.
  4. "Create API key in new project" 또는 "Create API key in existing project" 팝업이 나타납니다.
    • **처음이라면:** "Create API key in new project"를 클릭하세요. 자동으로 새 프로젝트가 만들어집니다.
    • **기존 사용자라면:** 목록에서 원하는 프로젝트를 선택합니다.
  5. 생성된 API 키 (`AIzaSy...` 로 시작하는 긴 문자열) 우측의 복사 아이콘을 눌러 복사합니다. (이것이 **Gemini 키**입니다.)

Step 3. '감린이' 앱에 API 키 입력하기

발급받은 키를 '감린이' 앱에 등록하면 모든 준비가 끝납니다.

  • 앱 접속 시 나타나는 최초 인증 팝업에 **Step 2에서 발급받은 Gemini API 키**를 붙여넣습니다.
  • (팝업을 껐다면) 우측 상단 '설정(⚙️)' > 'API 키 변경/설정'에서 입력할 수 있습니다.

⚠️API 키는 당신의 비밀번호입니다

  • 공유 금지: API 키를 타인과 공유하거나, GitHub, 블로그 등 공개된 장소에 절대 업로드하지 마세요.
  • 무료 한도: 무료 API는 1분당 요청 횟수가 제한됩니다. 채점/힌트 버튼을 너무 빠르게 연타하면 일시적으로 오류가 발생할 수 있습니다. 1분 후 다시 시도하세요.

📖사용 흐름 가이드

  1. 범위 선택: '단원 선택'과 '문제 필터'(예: '오늘의 복습', '80점 미만', '60점 미만')를 조합하여 학습할 범위를 정합니다.
  2. 시작 방식: '학습 시작', '랜덤 문제', 또는 '플래시카드' 버튼으로 원하는 학습 모드를 시작합니다.
  3. 문제 풀이: 답안을 입력합니다. 키보드로 타이핑하거나, 스마트폰 키보드의 음성 입력 기능을 활용하여 말로 답안을 입력할 수 있습니다.
  4. 보조 기능: 막히면 '힌트(단축키: H)' 버튼을 활용하고, 이전에 썼던 답안은 '이전 답안 불러오기(L)' 버튼으로 참고하거나 지울 수 있습니다.
  5. 채점 및 확인: '채점하기(단축키: Enter/G)' 버튼을 눌러 AI의 점수와 피드백, 모범 답안을 확인합니다.
  6. 복습 관리: '복습 추가(★)' 또는 '복습 제외(➖)' 버튼으로 문제의 복습 상태를 관리합니다. (두 버튼은 상호 배타적으로 작동합니다)
  7. 탐색: '이전(Ctrl+←)'/'다음(Ctrl+→)' 버튼으로 문제를 넘기거나, 하단 현황판의 번호 버튼, 또는 우측 '문제 목록'을 클릭하여 특정 문제로 바로 이동합니다.
  8. 설정 및 기타: 우측 상단 '설정(⚙️)', '업적(🏆)', '리포트(📊)' 아이콘을 통해 AI 모델 변경, 학습 성과 확인 등 고급 기능을 이용하세요.

🎙️말터디 음성 입력 활용법 (권장)

타이핑이 불편하거나 시간이 오래 걸리는 서술형 답안은 음성으로 입력하면 훨씬 빠르고 편리합니다. 별도의 API 키 없이도 스마트폰 키보드의 기본 음성 입력 기능을 사용하면 됩니다.

📱 모바일 키보드 음성 입력 사용법

💡 가장 쉽고 빠른 방법

  1. 답안 입력 영역을 터치하여 키보드를 띄웁니다.
  2. 키보드 하단의 마이크(🎤) 아이콘을 탭합니다.
  3. 음성 인식 모드가 시작되면 답안을 또박또박 말합니다.
  4. 말이 끝나면 자동으로 텍스트로 변환되어 입력됩니다.

🎯 Gboard에 회계감사 전문용어 등록하기 (강력 추천!)

안드로이드 기기에서 구글 Gboard를 사용한다면, 회계감사 전문 용어를 개인 사전에 미리 등록해두는 것이 음성 인식률을 크게 높이는 가장 효과적인 방법입니다.

⚡ 빠른 방법: 파일로 한 번에 가져오기

전문 용어를 하나씩 입력하기 번거롭다면, 미리 준비된 사전 파일을 다운로드하여 한 번에 가져올 수 있습니다!

  1. dictionary.txt 파일을 다운로드합니다. (우클릭 > 다른 이름으로 링크 저장)
  2. Gboard 설정 > 사전 > 개인 사전 > 한국어로 이동합니다.
  3. 우측 상단 메뉴(⋮)를 탭하고 "가져오기"를 선택합니다.
  4. 다운로드한 dictionary.txt 파일을 선택합니다.
  5. 40개 이상의 회계감사 전문용어가 자동으로 등록됩니다! 🎉

개별 등록 방법 (수동)

파일 가져오기가 안 되는 경우, 아래 방법으로 수동 등록할 수 있습니다.

  1. 설정 앱 열기: 안드로이드 기기의 '설정' 앱을 엽니다.
  2. Gboard 설정 찾기:
    • '시스템' > '언어 및 입력' > 'Gboard' 또는
    • '일반 관리' > '언어 및 입력' > '화상 키보드' > 'Gboard'로 이동
  3. 개인 사전 진입: 'Gboard 설정' 화면에서 '사전' > '개인 사전'을 선택합니다.
  4. 언어 선택: '한국어' 또는 사용 중인 언어를 선택합니다.
  5. 단어 추가: 우측 상단의 '+' (플러스) 버튼을 탭합니다.
  6. 단어 입력:
    • 단어: 등록할 전문 용어를 입력 (예: 중요왜곡표시위험)
    • 단축키: 선택 사항 (비워두거나, 짧은 단축어 입력 가능)
  7. 저장: 완료 또는 체크 표시를 눌러 저장합니다.

  • 부정위험

* 자주 나오는 용어를 미리 등록해두면, 음성 입력 시 해당 단어의 인식률이 크게 향상됩니다!

📝 기타 음성 입력 팁

  • 또박또박 발음: 빠르게 말하기보다는 명확하게 발음하는 것이 인식률을 높입니다.
  • 조용한 환경: 주변 소음이 적을수록 인식 정확도가 높아집니다.
  • 문장 단위 입력: 너무 긴 답안은 여러 문장으로 나눠서 입력하는 것이 좋습니다.
  • 인식 오류 수정: 음성 인식 후 잘못 입력된 부분은 키보드로 직접 수정하세요.

📊AI 학습 리포트 기능

'감린이'는 여러분의 학습 데이터를 기반으로 **AI가 생성하는 맞춤형 심층 리포트**를 제공합니다. 리포트는 4개의 탭으로 구성되어 있으며, 각 탭은 다양한 관점에서 학습 성과를 분석합니다.

📈 탭 1: 요약 및 차트 분석

학습 점수 추이를 **이동평균선 차트**로 시각화하여 단기적 노이즈를 제거하고 진짜 실력 추세를 확인할 수 있습니다.

  • 4대 이동평균선: 5일선(단기 컨디션), 20일선(중기 실력), 60일선(장기 기초체력), 총누적평균선
  • 골든 크로스: 단기선이 중장기선을 뚫고 올라가면 상승 추세 신호
  • 데드 크로스: 단기선이 중장기선 아래로 떨어지면 슬럼프 경고 신호
  • 정배열: 5일선 > 20일선 > 60일선 순서는 최상의 학습 상태
  • 지지선/저항선: 이평선이 방어선 또는 돌파해야 할 성장의 벽 역할

💡 Tip: 20일선(중기 실력선)의 우상향 각도가 가장 중요한 지표입니다. 일시적인 점수 등락에 일희일비하지 말고, 중장기 추세에 집중하세요!

🤖 탭 2: AI 심층 분석

Gemini AI가 여러분의 누적된 학습 기록을 종합적으로 분석하여 **3가지 핵심 인사이트**를 제공합니다.

  • 오답 패턴 분석: 자주 틀리는 문제 유형, 단원, 개념을 AI가 자동으로 분류
  • 개념 약점 진단: 점수가 낮은 영역의 공통 원인을 AI가 추론하고 보완 방향 제시
  • 성과 추이 평가: 최근 학습 효과성을 평가하고 개선 전략 제안

💡 Tip: AI 심층 분석은 최소 20문제 이상 풀었을 때 의미 있는 결과를 얻을 수 있습니다. 데이터가 많을수록 분석의 정확도가 높아집니다!

✅ 탭 3: 액션 플랜

AI 분석 결과를 바탕으로 **즉시 실행 가능한 구체적인 학습 계획**을 자동 생성합니다.

  • 우선순위 학습 단원: 가장 시급하게 보완해야 할 단원 순위
  • 추천 복습 문제: 약점 보완에 효과적인 문제 세트 자동 추출
  • 학습법 개선 제안: 현재 학습 패턴의 문제점과 개선 방향
  • 주간 학습 목표: 실현 가능한 단기 목표 설정

💡 Tip: 액션 플랜은 매주 리포트를 확인하여 지속적으로 업데이트하는 것이 효과적입니다!

📅 탭 4: 일일 학습 기록 & 암기팁

날짜별 학습 활동을 상세히 기록하고, **AI가 생성한 개인 맞춤형 암기팁**도 함께 확인할 수 있습니다.

일일 학습 기록:

  • 날짜별 풀이 문제 수, 평균 점수, 소요 시간
  • 학습 스트릭 (연속 학습 일수)
  • 주간/월간 통계 요약

💡 암기팁 기능:

AI가 각 문제의 핵심 개념을 분석하여 **기억하기 쉬운 암기 팁**을 자동 생성합니다.

  • 사용 방법: 플래시카드 모드에서 '암기팁 보기' 버튼 클릭
  • 내용: 핵심 키워드, 연상 기법, 구분법, 실무 예시 등
  • 장점: 단순 암기가 아닌 이해 기반 학습 유도
  • 활용: 틀린 문제나 헷갈리는 개념 위주로 암기팁 활용

💡 Tip: 암기팁은 플래시카드 모드에서 활용하면 가장 효과적입니다. 답을 확인한 후 암기팁을 읽고 핵심을 정리하세요!

🎯오늘의 복습 기능 & HLR 알고리즘

'감린이'는 **망각 곡선 이론**과 **개인별 학습 패턴 분석**을 결합하여, 지금 복습하면 가장 효과적인 문제들을 자동으로 추천합니다.

📚 복습 전략 선택

필터 영역의 '오늘의 복습' 섹션에서 다양한 우선순위 전략을 선택할 수 있습니다.

🧠 지능형 추천 (기본)

복습 플래그 > 미풀이 > 낮은 점수 > 오래된 순으로 종합 정렬. 균형 잡힌 기본 전략입니다.

🎯 HLR 기반 (회상 확률) ⭐

망각 곡선 알고리즘으로 **회상 확률이 낮은 순**(망각 임박)으로 정렬. 가장 과학적인 방법!

📉 점수 낮은 순

평균 점수가 낮은 문제부터 우선 복습. 약점 집중 공략에 효과적입니다.

⏰ 최근/오래전 틀린 순

🧬 HLR 알고리즘 (Half-Life Regression) 상세 설명

HLR은 **개인별 망각 곡선**을 듀오링고 모델 등 최신 암기 및 기억 관련 논문들을 참조, 수학적으로 모델링하여, 각 문제를 언제 복습하면 가장 효율적인지 예측하는 알고리즘입니다.

HLR-Lite 알고리즘 핵심 작동 원리:

📊 1단계: 회상 확률(p_recall) 계산

"지금 이 문제를 기억할 확률"을 수학 공식으로 추정합니다.

p_recall = 2(-경과일수 / 반감기)

예: 반감기가 10일인 문제를 10일 후에 다시 보면, 회상 확률은 50%입니다.

🧬 2단계: 반감기(h) 추정 및 갱신

각 문제의 학습 이력을 분석하여 "지식의 수명(반감기)"을 지속적으로 업데이트합니다.

새 반감기 = (기존 반감기 + 학습 간격) × 점수 배율
  • • 초기값: h = 1.0일 (첫 학습 시)
  • • 학습 간격(delta): 이전 학습과 이번 학습 사이의 일수
  • • 점수 배율(factor):
    • - 90점 이상: 2.0~3.0배 (매우 잘 외움)
    • - 70~89점: 1.4~1.9배 (잘 외움)
    • - 50~69점: 1.1배 (보통)
    • - 50점 미만: 0.6배 (감소, 다시 복습 필요)

🎯 3단계: 시뮬레이션 예시

Day 0 (첫 학습, 80점):

h = (1.0 + 1.0) × 1.65 = 3.3일

Day 5 (5일 후 복습, 95점):

h = (3.3 + 5.0) × 2.5 = 20.75일

→ 지식이 강화되어 반감기가 크게 증가!

Day 15 (10일 후 복습, 40점):

h = (20.75 + 10.0) × 0.6 = 18.45일

→ 점수가 낮아 반감기가 감소. 더 자주 복습 필요

⚙️ 추가 우선순위 요소:

  1. 복습 플래그 최우선: 사용자가 수동으로 '복습 추가(★)'한 문제는 HLR 점수와 관계없이 최상위 우선순위
  2. FSRS 난이도 보정: 플래시카드에서 체크한 '쉬움/보통/어려움' 난이도가 우선순위 계산에 추가 반영됩니다
  3. 평균 점수 고려: 전체 풀이 이력의 평균 점수가 낮을수록 우선순위 상승
  4. 최근 재인 방지: 24시간 이내에 본 카드는 우선순위를 낮춰서 과도한 반복 방지

💳 플래시카드 난이도 평가와 복습 연동

플래시카드 모드에서 답을 확인한 후, **난이도를 체크**하면 HLR 알고리즘이 이를 학습하여 다음 복습 추천에 반영합니다.

난이도 평가 사용 방법:

  1. 플래시카드 모드 진입: 우측 상단 '플래시카드' 버튼 클릭
  2. 답변 확인: '답변 보기' 버튼으로 정답 확인
  3. 난이도 선택: 😊 쉬움 / 🤔 보통 / 😰 어려움 / ⏭️ 건너뛰기 중 선택
  4. 자동 반영: 선택한 난이도가 FSRS 점수로 변환되어 HLR 알고리즘에 즉시 적용
  5. 다음 복습: '오늘의 복습 - HLR 기반' 필터를 사용하면 난이도가 반영된 우선순위로 문제가 정렬됩니다

키보드 단축키: 1(쉬움), 2(보통), 3(어려움), 0(건너뛰기)로 빠르게 평가할 수 있습니다!

🎯 HLR 복습 시스템 활용 꿀팁:

  • 매일 15분 투자: '오늘의 복습 - HLR 기반'으로 10~15문제만 빠르게 플래시카드로 복습
  • 난이도는 솔직하게: 실제로 기억하기 어려웠다면 '어려움'으로 체크. 알고리즘이 더 정확해집니다!
  • 복습 제외 활용: 이미 완벽히 외운 문제는 '복습 제외(X)'로 체크하여 효율 극대화
  • 장기적 관점: HLR은 최소 2주 이상 사용했을 때 진가를 발휘합니다. 꾸준히 사용하세요!
  • 복습 기준 설정: 설정 모달에서 'HLR 기반'을 선택하면 캘린더/통계에도 HLR 기준이 적용됩니다

🎤 앱 내장 음성 입력 (STT) 기능

⚠️ 대부분의 경우 모바일 키보드 기본 음성 입력을 사용하는 것이 더 편리합니다

위에서 안내한 Gboard 개인 사전 등록 방법이 음성 인식률도 더 높고, 별도의 API 키 발급 없이 바로 사용할 수 있어 권장됩니다. 이 앱 내장 STT 기능은 특수한 경우에만 사용하세요.

Step 1. Google STT API 키 발급 (음성 입력용)

이 키는 AI의 '귀(👂)' 역할을 하며, 사용자의 음성을 텍스트로 변환해주는 음성 입력 기능에만 사용됩니다.

🚨 주의: 이 키는 Gemini 키와는 완전히 다른 별개의 키입니다.

  1. [**Google Cloud Console**]에 접속하여 Google 계정으로 로그인합니다.
    (결제 계정 등록이 필요할 수 있으나, STT는 매월 60분의 무료 사용량을 제공합니다.)
  2. 페이지 상단에서 프로젝트를 선택하거나 [새 프로젝트]를 만듭니다. (Gemini 키와 다른 프로젝트여도 상관없습니다.)
  3. 왼쪽 상단 메뉴(☰) > **[API 및 서비스]** > **[라이브러리]**로 이동합니다.
  4. 검색창에 "Cloud Speech-to-Text API"를 검색하고, 검색된 API를 선택한 후 **[사용 설정]** 버튼을 클릭합니다.
  5. 왼쪽 메뉴에서 **[API 및 서비스]** > **[사용자 인증 정보]**로 이동합니다.
  6. 페이지 상단의 **[+ 사용자 인증 정보 만들기]** 버튼을 클릭하고, 드롭다운 메뉴에서 **[API 키]**를 선택합니다. (절대 '서비스 계정'이나 'OAuth'를 선택하지 마세요.)
  7. 팝업창에 `AIzaSy...`로 시작하는 **STT API 키**가 생성됩니다. 이 키를 복사합니다.

Step 2. '감린이' 앱에 STT 키 입력하기

  1. 앱 우측 상단 '설정(⚙️)' 아이콘을 클릭합니다.
  2. "음성 인식(STT) 설정" 섹션으로 스크롤합니다.
  3. 공급자를 "Google Cloud STT (권장)"로 선택합니다.
  4. "Google STT API Key" 입력란에 **Step 1에서 발급받은 STT 키**를 붙여넣습니다.
  5. 설정 모달 하단의 [닫기] 버튼을 누릅니다.
  6. 이제 답안 입력 영역 옆에 **음성 입력(🎤) 버튼**이 활성화됩니다.

💡 앱 내장 STT 기능의 장점

  • 회계감사 전문 용어에 대한 '키워드 부스팅' 적용
  • PC 환경에서도 음성 입력 가능
  • 앱 내에서 바로 녹음/변환 가능
