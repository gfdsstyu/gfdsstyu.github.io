2

Gemini API 키 발급 (필수) 중요

AI 채점 기능을 사용하려면 Google AI Studio에서 무료 API 키를 발급받아야 합니다.

📋 API 키 발급 단계별 가이드

  1. Google AI Studio에 접속하여 구글 계정으로 로그인합니다.
  2. 처음 접속하는 경우:
    • "Create API key" 또는 "Get API key" 버튼이 보이면 바로 클릭
    • 프로젝트가 없다면 "Create API key in new project" 옵션을 선택하여 새 프로젝트 자동 생성
    • 이미 Google Cloud 프로젝트가 있다면 기존 프로젝트 선택 가능
  3. 좌측 메뉴에서 "Get API key" 클릭 (두 번째 접속부터는 이 메뉴 이용)
  4. "Create API key" 버튼을 눌러 API 키 생성
    • 생성된 키는 AIzaSy...로 시작하는 긴 문자열입니다
    • 복사 버튼을 눌러 클립보드에 저장하세요
  5. 감린이 웹사이트로 돌아와 우측 상단 ⚙️ 설정 → API 키 관리에 붙여넣기
💡

처음이라 어렵게 느껴지시나요?

프로젝트 생성이 처음이더라도 걱정 마세요! "Create API key in new project" 버튼 하나로 프로젝트와 API 키가 자동으로 만들어집니다.

🤖 AI 모델 & 감린이 챗봇 2.0

⚡ Groq AI 모델 추가 NEW

Groq은 초고속 AI 추론 플랫폼으로, 무료이면서도 빠른 응답 속도를 제공합니다.

✅ 장점
  • • 무료: API 키 무료 발급 가능
  • • 빠른 속도: 빠른 응답 시간
  • • API 발급: console.groq.com에서 발급
⚠️ 제한사항
  • • 기준서 채점 전용: 낮은 TPM (분당 토큰) 한도로 인해 Exam 모드에서는 사용 불가
  • • 단답형 문제: 짧은 답안에 최적화

💡 추천 사용법: 기준서 모드에서 빠르게 채점받고 싶을 때, Exam 모드에서는 Gemini 사용을 권장합니다.

• 학습 기록 저장: 대화 내용이 자동 저장되어 나중에 다시 볼 수 있음

📝 Exam 모드

• 우측 하단 파비콘 숨김: 실전 시험 환경을 위해 플로팅 버튼 미표시

• 유일한 진입점: 채점 결과 화면의 각 문제별 "감린이로 깊이 학습하기" 버튼

• 문제 컨텍스트 자동 주입: 해당 문제의 지문, 모범답안, 채점평 모두 기억

• 물음 선택 드롭다운: 다른 물음으로 컨텍스트 전환 가능

• 빠른 질문: "오답 원인 분석", "올바른 접근법", "암기법" 등 시험 전용 버튼

🧠 스마트 복습 (HLR 알고리즘)

"언제 복습해야 할까?" 고민하지 마세요. 망각 곡선 이론(Half-Life Regression)을 적용해, 기억에서 사라지기 직전인 문제를 우선적으로 추천합니다.

📋 다양한 복습 기준 제공

🧠 HLR 기반 (지능형 추천)

망각 확률을 계산하여 지금 까먹기 직전인 문제를 우선 추천합니다. 가장 효율적인 복습 방법입니다.

📉 점수 낮은 순

내가 가장 약한 문제부터 집중 공략합니다. 약점을 빠르게 보완하고 싶을 때 사용하세요.

🕐 최근 푼 순

방금 풀었던 문제를 다시 확인할 때 유용합니다. 오늘 공부한 내용을 빠르게 복습하세요.

⏰ 오래된 순

오랫동안 안 풀어본 문제를 찾아줍니다. 장기 기억 전환을 위한 복습에 효과적입니다.

🔬 HLR 알고리즘이란?

Half-Life Regression (반감기 회귀)는 에빙하우스의 망각곡선 이론을 개인별로 맞춤화한 지능형 알고리즘입니다.

1️⃣ 난이도 기록

문제를 풀 때마다 체감 난이도(쉬움/보통/어려움)를 기록합니다.

2️⃣ 망각 확률 계산

시간 경과, 난이도, 반복 횟수를 종합해 현재 기억 확률을 예측합니다.

3️⃣ 최적 타이밍 추천

망각 직전(회상 확률 20~40%)인 문제를 우선 보여줍니다.

💡 플래시카드 모드에서 사용하기

정답 확인 후, 솔직하게 난이도를 평가하세요. 이 데이터가 알고리즘의 핵심입니다.

😊 쉬움 (1) 🤔 보통 (2) 😰 어려움 (3)

✅ 쉬움: 다음 복습까지 간격이 길어집니다
🔄 보통: 적절한 간격으로 다시 등장합니다
⚠️ 어려움: 빠른 시간 내에 다시 복습하게 됩니다

💡 설정에서 '복습 기준'을 [HLR 기반]으로 선택하면 가장 강력한 효과를 발휘합니다.

🎙️ 말터디 (음성 입력)

긴 서술형 답안, 타자 치지 말고 말하세요. 모바일 기기의 키보드 음성 입력을 활용하는 것이 가장 효율적입니다.

📱 모바일 활용 (권장)

  1. 키보드를 띄우고 마이크 아이콘(🎤) 터치
  2. 답안을 말하면 텍스트로 자동 변환
  3. Tip: '중요왜곡표시위험' 등 전문 용어를 키보드 단축어에 등록하면 인식률 급상승!

💾 로그인 없이 사용하는 경우 (수동 백업 필수)

로그인 없이 사용하면 데이터가 브라우저에만 저장됩니다. 소중한 답안과 AI 피드백을 잃지 않으려면 수동 백업이 필요합니다:

  • 데이터 내보내기: [설정(⚙️)] → [데이터 관리] → [📥 데이터 내보내기]로 백업 파일을 저장
  • 데이터 가져오기: 새 기기에서 [📤 데이터 가져오기]를 통해 기록 복원
  • 데이터 병합: 양쪽 기기의 기록을 합치려면 [🔀 데이터 병합 가져오기] 사용

🔍 로그인 상태 확인

• 우측 상단 메뉴에서 로그인 상태를 확인하고, 올바른 계정으로 로그인되어 있는지 확인하세요.

⚠️ 비로그인 상태로 사용한 경우

• 로그인을 하지 않으면 데이터가 브라우저에만 저장됩니다.

• 브라우저 캐시 삭제, 시크릿 모드 종료, 브라우저 재설치 등의 경우 데이터가 사라질 수 있습니다.

💡 백업한 데이터가 있다면 [데이터 가져오기] 기능으로 복원하세요.
