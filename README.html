⚠️면책 조항 (Disclaimer)

본 프로그램은 개인 학습 목적으로 제작되었습니다.

  • 정확성 미보장: AI 모델(Gemini)이 제공하는 채점 점수, 피드백 및 기타 모든 정보는 부정확할 수 있으며, 어떠한 경우에도 학술적, 법적 근거로 사용될 수 없습니다.
  • 보조 도구: 이 도구는 암기를 위한 '보조 수단'일 뿐이며, AI의 채점 결과가 사용자의 실제 이해도를 완벽하게 반영하지 않습니다.
  • 최종 판단: 모든 내용의 최종적인 정확성 판단과 학습에 대한 책임은 전적으로 사용자 본인에게 있습니다.
  • API 책임: API 키 사용으로 인해 발생하는 모든 책임(비용, 한도 초과 등)은 사용자 본인에게 있습니다.

이 프로그램은 회계감사 서술형 문제를 효과적으로 암기하고 테스트하기 위해 만들어진 개인 학습 도구입니다. Gemini AI 모델을 활용하여 사용자의 서술형 답안을 모범 답안과 비교하여 유사도 점수와 AI 피드백을 제공합니다. 별도의 설치 없이 웹사이트에서 바로 이용할 수 있습니다.

🧰주요 기능 (v3.2)

  • 🤖

    AI 자동 채점 (모델 선택 가능)

    사용자의 답안을 모범 답안과 비교하여 0~100점 척도의 점수와 함께 AI의 총평(피드백)을 제공합니다. 설정에서 Gemini Flash (기본) 또는 Flash-Lite 모델을 선택할 수 있습니다.

  • 💡

    AI 힌트 제공

    문제 풀이 중 막힐 때 '힌트' 버튼을 누르면, 정답을 직접 노출하지 않으면서 핵심 개념을 떠올릴 수 있도록 AI가 도움말을 제공합니다.

  • 📚

    단원별 학습 및 랜덤 문제

    특정 단원을 선택하여 집중 학습하거나, '랜덤 문제' 버튼으로 현재 필터 조건 내에서 무작위 문제를 풀 수 있습니다.

  • 📝

    다양한 문제 필터 (오늘의 복습 포함)

    '아직 안 푼 문제', '80점 미만', '60점 미만' 등 오답노트 필터와 함께, 점수가 낮거나 학습한 지 오래된 문제를 추천해주는 '오늘의 복습 (추천)' 필터를 제공합니다.

  • 📊

    학습 현황판 (정보 확장)

    단원별 평균 점수, 응시율과 더불어 평균 회독 수, 최근 학습일 정보를 제공합니다. 문제별 점수를 색상으로 한눈에 파악하고, 번호 버튼을 클릭하여 해당 문제로 즉시 이동할 수 있습니다.

  • 📄

    이전 답안 불러오기 / 지우기

    답안 입력창 하단의 버튼으로 이전에 저장했던 답안을 불러오거나, 현재 입력된 내용을 지울 수 있습니다.

  • 🎨

    다크 모드 지원 (시스템 설정 연동)

    설정에서 라이트/다크 모드를 선택하거나, 사용자의 OS 설정을 자동으로 따라가는 테마를 적용할 수 있습니다.

  • 💾

    학습 기록 자동 저장 (로컬)

    모든 채점 결과(점수, 피드백, 답안, 학습일시 등)는 사용자의 웹 브라우저(`localStorage`)에 자동으로 저장됩니다.

  • 🔄

    데이터 백업/복원 및 초기화

    설정에서 모든 학습 기록과 설정을 JSON 파일로 내보내거나 가져올 수 있습니다. 다른 PC나 브라우저로 학습 환경을 옮길 때 유용합니다.

1. 웹사이트 접속

별도의 설치 없이 아래 주소로 접속하면 바로 사용할 수 있습니다.

2. Gemini API 키 발급 (상세 가이드)

'채점하기'와 '힌트보기' 기능을 사용하려면 사용자 개인의 **Gemini API 키**가 반드시 필요합니다. 무료 등급에서 일정 한도 내에서 무료로 사용할 수 있습니다.

  1. [**Google AI Studio**] 사이트에 접속하여 Google 계정으로 로그인합니다.
  2. 서비스 약관 동의 팝업이 나타나면 동의하고 계속 진행합니다.
  3. 좌측 메뉴에서 **Get API key (API 키 받기)**를 클릭합니다.
  4. **"Create API key" (API 키 만들기)** 버튼을 클릭합니다.

    이때, 키를 저장할 Google Cloud 프로젝트를 선택하라는 팝업이 나타날 수 있습니다. 기존 프로젝트가 없다면 **"Create a new project" (새 프로젝트 만들기)**를 선택하여 간단히 프로젝트를 생성하면 됩니다.

  5. 생성된 API 키 (`AIzaSy...` 로 시작하는 긴 문자열) 우측의 복사 아이콘을 눌러 복사합니다. 이 키를 감린이 앱에서 사용합니다.

⚠️API 키 사용 시 유의사항 (중요)

  • 보안: API 키는 개인 비밀번호와 같습니다. **절대 이 키를 타인과 공유하거나, GitHub, 블로그 등 공개된 장소에 업로드하지 마세요**.
  • 무료 한도 (Rate Limit): 무료 등급 API는 1분당 요청 횟수(RPM)가 제한되어 있습니다. `채점하기` 또는 `힌트` 버튼을 너무 짧은 시간 안에 여러 번 클릭하면, API가 일시적으로 "429 (Too many requests)" 오류를 반환할 수 있습니다. 이 경우, 1분 정도 기다린 후 다시 시도해야 합니다.
  • 비용: 이 프로그램은 무료 등급을 사용하도록 설계되었으나, Google 정책 변경이나 사용량 한도 초과 시 비용이 발생할 수 있습니다. 개인 학습 용도로만 사용해 주세요.

📖3. 프로그램 사용 방법 (단계별)

  1. API 키 입력 (최초 실행 시):

    웹사이트에 처음 접속하면 API 키 입력 팝업이 나타납니다. 위에서 발급받은 Gemini API 키를 붙여넣고 '저장' 버튼을 누릅니다. `이 브라우저에 키 저장` 체크박스를 선택하면, 다음부터 이 PC의 브라우저에서는 키를 다시 입력할 필요가 없습니다. (공용 PC에서는 체크 해제 권장)

    API 키 변경은 우측 상단 '설정(⚙️)' 메뉴에서 가능합니다.

  2. 학습 범위 선택:

    단원 선택: 특정 단원만 공부하고 싶다면 드롭다운에서 해당 단원을 선택합니다. (기본값: 전체 단원)
    문제 필터: 복습을 위해 '오늘의 복습', '80점 미만' 등을 선택합니다. (기본값: 모든 문제)

  3. 학습 시작 / 랜덤 문제:

    `학습 시작` 버튼을 클릭하면 선택한 범위의 첫 문제부터 순서대로 풀이가 시작됩니다. `랜덤 문제` 버튼을 누르면 선택한 범위 내에서 무작위 문제가 즉시 표시됩니다.

  4. 답안 입력 및 채점/힌트:

    답안 입력란에 암기한 내용을 작성합니다. 필요시 `힌트` 버튼을 눌러 도움을 받을 수 있습니다. 작성이 끝나면 `채점하기` 버튼을 클릭합니다.

    답안 입력란 아래 `이전 답안 불러오기` 버튼으로 마지막으로 저장된 답안을 불러오거나, 불러온 답안을 다시 지울 수 있습니다.

  5. 결과 확인:

    채점이 완료되면 하단에 **채점 결과**가 나타납니다. `유사도 점수`, `🤖 AI 총평`, `📘 모범 답안`을 비교하며 학습합니다.

  6. 문제 이동:

    `이전` / `다음` 버튼을 눌러 문제 사이를 이동합니다.

🌟4. 핵심 기능 상세

💡 오늘의 복습 (추천) 필터

'문제 필터'에서 '오늘의 복습'을 선택하면, AI가 사용자의 학습 기록을 분석하여 지금 시점에 복습하면 가장 효율적인 문제들을 **최대 10개**까지 추천해줍니다. 추천 알고리즘은 다음 우선순위를 따릅니다:

  1. 아직 한 번도 풀지 않았거나, 60점 미만이거나, 마지막으로 푼 지 7일 이상 지난 문제들을 후보로 선정합니다.
  2. 선정된 문제들을 **점수가 낮은 순**으로, 점수가 같다면 **가장 오래 전에 푼 순**으로 정렬합니다.
  3. 정렬된 순서대로 상위 10개의 문제를 보여줍니다.

학습 현황판 (Dashboard)
