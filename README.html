🧠 스마트 복습 (HLR 알고리즘)

"언제 복습해야 할까?" 고민하지 마세요. 망각 곡선 이론(Half-Life Regression)을 적용해, 기억에서 사라지기 직전인 문제를 우선적으로 추천합니다.

📋 다양한 복습 기준 제공

🧠 HLR 기반 (지능형 추천)

망각 확률을 계산하여 지금 까먹기 직전인 문제를 우선 추천합니다. 가장 효율적인 복습 방법입니다.

📉 점수 낮은 순

내가 가장 약한 문제부터 집중 공략합니다. 약점을 빠르게 보완하고 싶을 때 사용하세요.

🕐 최근 푼 순

방금 풀었던 문제를 다시 확인할 때 유용합니다. 오늘 공부한 내용을 빠르게 복습하세요.

⏰ 오래된 순

오랫동안 안 풀어본 문제를 찾아줍니다. 장기 기억 전환을 위한 복습에 효과적입니다.

🔬 HLR 알고리즘이란?

Half-Life Regression (반감기 회귀)는 에빙하우스의 망각곡선 이론을 개인별로 맞춤화한 지능형 알고리즘입니다.

1️⃣ 난이도 기록

문제를 풀 때마다 체감 난이도(쉬움/보통/어려움)를 기록합니다.

2️⃣ 망각 확률 계산

시간 경과, 난이도, 반복 횟수를 종합해 현재 기억 확률을 예측합니다.

3️⃣ 최적 타이밍 추천

망각 직전(회상 확률 20~40%)인 문제를 우선 보여줍니다.

💡 플래시카드 모드에서 사용하기

정답 확인 후, 솔직하게 난이도를 평가하세요. 이 데이터가 알고리즘의 핵심입니다.

😊 쉬움 (1) 🤔 보통 (2) 😰 어려움 (3)

✅ 쉬움: 다음 복습까지 간격이 길어집니다
🔄 보통: 적절한 간격으로 다시 등장합니다
⚠️ 어려움: 빠른 시간 내에 다시 복습하게 됩니다

💡 설정에서 '복습 기준'을 [HLR 기반]으로 선택하면 가장 강력한 효과를 발휘합니다.

🎙️ 말터디 (음성 입력)

긴 서술형 답안, 타자 치지 말고 말하세요. 모바일 기기의 키보드 음성 입력을 활용하는 것이 가장 효율적입니다.

📱 모바일 활용 (권장)

  1. 키보드를 띄우고 마이크 아이콘(🎤) 터치
  2. 답안을 말하면 텍스트로 자동 변환
  3. Tip: '중요왜곡표시위험' 등 전문 용어를 키보드 단축어에 등록하면 인식률 급상승!

🖥️ PC 내장 기능 (고급)

Google Cloud STT API 키가 있다면 설정에서 등록하여 PC에서도 고품질 음성 인식을 쓸 수 있습니다. (설정 > 실험실 기능)

🤖 안드로이드 사용자 추천: Gboard

안드로이드에서는 Google 키보드(Gboard) 사용을 강력히 추천합니다. 회계감사 전문 용어 인식률이 삼성 키보드 대비 월등히 높습니다.

✅ 설치: Play 스토어에서 "Gboard" 검색 후 설치

✅ 활성화: 설정 > 언어 및 입력 > 가상 키보드 > Gboard 선택

✅ 음성 입력: Gboard 사용 시 마이크 버튼 눌러 음성 입력 시작

📖GBoard(안드로이드) 전용 용어 사전

음성 인식률을 높이기 위한 회계감사 전문 용어 목록을 제공합니다. 다음 파일을 GBoard 에 적용시 음성인식률이 높아집니다.

📥 dictionary.txt 다운로드 (중요왜곡표시위험, 계속기업, 특수관계자 등 200+ 용어 수록)

🔍 KAM 사례형 훈련

핵심감사사항(KAM)은 최근 CPA 시험의 핵심입니다. 실제 사례를 분석하고, Why(선정 이유)와 How(감사절차)를 단계별로 훈련하세요.

Step 1: Why (선정 이유)

📋 실제 사례 제시: 건설업 수익인식, 금융상품 평가 등

💭 서술형 작성: 위험의 원천과 재무적 중요성 포함

🤖 AI 피드백: 제출 즉시 핵심 키워드 누락 여부 분석

예시: "경영진의 추정에 고도의 판단이 개입되며..."

Step 2: How (감사절차)

💡 힌트 제공: Step 1 모범 답안을 참고

🔍 절차 서술: 내부통제 테스트, 문서 검사 등 구체화

⚠️ Gap Analysis: 누락된 절차를 빨간색으로 표시

⚠️ 전문가 활용 여부 언급 누락
⚠️ 문서 검사 절차 구체화 필요

랭킹 시스템: The Audit Ladder

매일의 꾸준함을 '업적 점수(Achievement Points, AP)'로 증명하세요. 문제를 풀고, 업적을 달성하며 AP를 쌓아 더 높은 티어에 도전하세요.

💎 AP 획득 방법

📝 문제 풀이 점수 (매일 채굴)

  • ✅ 75점 이상: +3 AP (고득점 보너스)
  • 📊 75점 미만: +1 AP (기본 점수)

💡 데일리 미션 보너스

- 10문제 달성: +30 AP 🎁
- 30문제 달성: +40 AP 🎁
- 50문제 달성: +50 AP 🎁

🏅 업적 달성 보너스

🥉

Bronze 업적

10~30 AP (예: 첫 80점, 3일 연속 학습)

🥈

Silver 업적

50~150 AP (예: 7일 연속, 단원 마스터)

🥇

Gold 업적

200~500 AP (예: 1회독 완료, 30일 연속)

🎖️ 티어 시스템 (총 누적 AP 기준)

총 누적 AP는 평생 획득한 AP의 합계입니다. 한번 획득한 AP는 절대 감소하지 않으며, 티어 승급의 기준이 됩니다.

💾 로그인 없이 사용하는 경우 (수동 백업 필수)

로그인 없이 사용하면 데이터가 브라우저에만 저장됩니다. 소중한 답안과 AI 피드백을 잃지 않으려면 수동 백업이 필요합니다:

  • 데이터 내보내기: [설정(⚙️)] → [데이터 관리] → [📥 데이터 내보내기]로 백업 파일을 저장
  • 데이터 가져오기: 새 기기에서 [📤 데이터 가져오기]를 통해 기록 복원
  • 데이터 병합: 양쪽 기기의 기록을 합치려면 [🔀 데이터 병합 가져오기] 사용
