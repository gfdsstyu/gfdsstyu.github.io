  • API 책임: API 키 사용으로 인해 발생하는 모든 책임(비용, 한도 초과 등)은 사용자 본인에게 있습니다. (상세 내용은 'API 키 사용 시 유의사항' 참고)

이 프로그램은 회계감사 서술형 문제를 효과적으로 암기하고 테스트하기 위해 만들어진 개인 학습 도구입니다. Gemini AI 모델을 활용하여 사용자의 서술형 답안을 모범 답안과 비교하여 유사도 점수와 AI 피드백을 제공합니다. 모든 기능은 단 하나의 `html` 파일 안에서 실행되며, 별도의 서버나 설치가 필요하지 않습니다.

🔑시작 전 필수 준비 사항: Gemini API 키

이 프로그램의 핵심 기능인 '채점하기'는 Google의 AI 모델(Gemini)을 호출하여 작동합니다. 이를 위해 사용자 개인의 **Gemini API 키**가 반드시 필요합니다. API 키는 무료 등급(Free Tier)에서 일정 한도 내에서 무료로 사용할 수 있습니다.

API 키 발급 방법

  1. [**Google AI Studio**] 사이트에 접속하여 Google 계정으로 로그인합니다.
  2. 로그인 후, 왼쪽 상단 또는 중앙의 **"Get API key" (API 키 받기)** 버튼을 클릭합니다.
  3. **"Create API key" (API 키 만들기)** 버튼을 클릭하여 새 키를 생성합니다.
  4. 생성된 API 키 (AIzaSy... 로 시작하는 긴 문자열)를 복사합니다. 이 키를 프로그램에서 사용합니다.

⚠️API 키 사용 시 유의사항 (중요)

  • 보안: API 키는 개인 비밀번호와 같습니다. **절대 이 키를 타인과 공유하거나, GitHub, 블로그 등 공개된 장소에 업로드하지 마세요.** (키가 노출되면 다른 사람이 내 한도를 소진시킬 수 있습니다.)
  • 무료 한도 (Rate Limit): 무료 등급 API는 1분당 요청할 수 있는 횟수(RPM)가 제한되어 있습니다. `채점하기` 버튼을 너무 짧은 시간 안에 여러 번 클릭하면(연속 클릭 등), API가 일시적으로 "429 (Too many requests)" 오류를 반환하며 작동을 멈출 수 있습니다. 이 경우, 1분 정도 기다린 후 다시 시도해야 합니다.
  • 비용: 이 프로그램은 무료 등급(Free Tier)을 사용하도록 설계되었으나, Google의 정책 변경이나 사용량 한도 초과 시 비용이 발생할 가능성이 있습니다. 개인 학습 용도로만 사용해 주세요.

🚀1. 파일 저장 및 실행 방법 (PC에서 열기)

이 프로그램은 .html 파일 1개로 제공되며, 사용자의 PC에서 직접 열어 사용합니다.

  1. 제공받은 html 파일 (예: ai_회계감사_암기_도우미_v3.html)을 컴퓨터의 바탕화면이나 원하는 폴더에 저장합니다.
  2. 저장한 html 파일을 더블클릭하면 크롬, 엣지, 사파리 등 사용자 PC의 기본 웹 브라우저에서 프로그램이 열립니다.

📖2. 프로그램 사용 방법 (단계별)

  1. API 키 입력:

    가장 먼저, 위에서 발급받은 Gemini API 키를 "🔑 Gemini API Key" 입력란에 붙여넣습니다. `이 브라우저에 키 저장` 체크박스를 선택하면, 다음부터 이 PC의 브라우저에서는 키를 다시 입력할 필요가 없습니다.

  2. 학습 범위 선택:

    단원 선택: 특정 단원만 공부하고 싶다면 드롭다운에서 해당 단원을 선택합니다. (기본값: 전체 단원)
    문제 필터: 복습을 위해 '80점 미만' 등을 선택합니다. (기본값: 모든 문제)

  3. 학습 시작:

    `학습 시작` 버튼을 클릭하면 선택한 범위의 문제 풀이가 시작됩니다.

  4. 답안 입력 및 채점:

    `여기에 정답을 입력하세요...` 란에 암기한 답안을 작성한 후, `채점하기` 버튼을 클릭합니다.

  5. 결과 확인:

    채점이 완료되면 하단에 **채점 결과**가 나타납니다. `유사도 점수`, `🤖 AI 총평`, `📘 모범 답안`을 비교하며 학습합니다.

  6. 문제 이동:

    `이전` / `다음` 버튼을 눌러 문제 사이를 이동합니다.

학습 기록 저장 및 초기화

  • 💾

    자동 저장

    `채점하기`를 누르는 순간, 모든 기록(점수, 피드백, 내가 쓴 답안)이 브라우저의 `localStorage`라는 공간에 **자동으로 저장**됩니다. 브라우저를 닫거나 컴퓨터를 재부팅해도 이 기록은 사라지지 않습니다.

  • 🔄

    기록 초기화

    모든 점수 기록을 지우고 처음부터 다시 시작하고 싶다면, 현황판 우측 상단의 `학습 기록 초기화` 버튼을 누르세요.
