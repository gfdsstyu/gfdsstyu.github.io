🚀 시작하기: 3분이면 충분합니다

Step 1. 웹사이트 접속

별도의 설치 없이 감린이 웹사이트에 접속하면 바로 시작할 수 있습니다.

Step 2. Gemini API 키 발급 (상세 가이드)

AI 기능을 사용하려면 사용자 개인의 **Gemini API 키**가 반드시 필요합니다. 무료로 간단히 만들 수 있습니다.

  1. [**Google AI Studio**]에 접속하여 Google 계정으로 로그인합니다. (약관 동의)
  2. 좌측 메뉴에서 **Get API key (API 키 받기)**를 클릭합니다.
  3. 파란색 **"Create API key" (API 키 만들기)** 버튼을 클릭합니다.
  4. "Create API key in new project" 또는 "Create API key in existing project" 팝업이 나타납니다.
    • **처음이라면:** "Create API key in new project"를 클릭하세요. 자동으로 새 프로젝트가 만들어집니다.
    • **기존 사용자라면:** 목록에서 원하는 프로젝트를 선택합니다.
  5. 생성된 API 키 (`AIzaSy...` 로 시작하는 긴 문자열) 우측의 복사 아이콘을 눌러 복사합니다.

이제 복사한 키를 감린이 웹사이트의 최초 인증 팝업에 붙여넣으면 모든 준비가 끝납니다!

⚠️API 키는 당신의 비밀번호입니다

  • 공유 금지: API 키를 타인과 공유하거나, GitHub, 블로그 등 공개된 장소에 절대 업로드하지 마세요.
  • 무료 한도: 무료 API는 1분당 요청 횟수가 제한됩니다. 채점/힌트 버튼을 너무 빠르게 연타하면 일시적으로 오류가 발생할 수 있습니다. 1분 후 다시 시도하세요.

📖사용 흐름 가이드

  1. 범위 선택: '단원 선택'과 '문제 필터'(예: '오늘의 복습', '80점 미만')를 조합하여 학습할 범위를 정합니다.
  2. 시작 방식: '학습 시작' 버튼으로 선택한 범위의 첫 문제부터 순서대로 풀거나, '랜덤 문제' 버튼으로 범위 내 무작위 문제를 즉시 풉니다.
  3. 문제 풀이: 답안을 입력합니다. 막히면 '힌트' 버튼을 활용하고, 이전에 썼던 답안은 '이전 답안 불러오기' 버튼으로 참고하거나 지울 수 있습니다.
  4. 채점 및 확인: '채점하기' 버튼을 눌러 AI의 점수와 피드백, 모범 답안을 확인합니다.
  5. 탐색: '이전'/'다음' 버튼으로 문제를 넘기거나, 하단 현황판의 번호 버튼을 클릭하여 특정 문제로 바로 이동합니다.
  6. 설정 활용: 우측 상단 '설정(⚙️)' 메뉴에서 AI 모델 변경, 다크 모드 전환, 학습 기록 백업/복원 등을 이용하세요.
