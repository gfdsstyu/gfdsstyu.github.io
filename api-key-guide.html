Google AI Studio API 키 발급 가이드

감린이에서 AI 채점 기능을 사용하기 위한 API 키 발급 방법

사전 준비사항

👤
Google 계정 - Gmail 등 Google 계정이 필요합니다
💳
결제 수단 (선택) - 무료 체험판 등록 시 카드 정보 필요 (자동 결제 없음)
💰
무료 사용량 - 무료 계정은 일일 약 20회(RPD)로 제한됨 (무료 체험판 등록 권장)

목차

  1. Google AI Studio 접속하기
  2. 프로젝트 생성하기
  3. 무료 체험판 등록하기 (선택)
  4. 무료 크레딧 확인하기
  5. API 키 발급받기
  6. 감린이에 API 키 등록하기

1 Google AI Studio 접속하기

아래 버튼을 클릭하여 Google AI Studio에 접속합니다.

Google AI Studio 바로가기
  1. Google AI Studio 웹사이트에 접속합니다
  2. 오른쪽 상단의 "로그인" 또는 "Sign in"을 클릭합니다
  3. Google 계정으로 로그인합니다
Tip: 이미 Google 계정에 로그인되어 있다면 자동으로 로그인됩니다.

2 프로젝트 생성하기

API 키를 발급받으려면 먼저 프로젝트를 생성해야 합니다.

  1. 왼쪽 메뉴에서 "프로젝트"를 클릭합니다
  2. 오른쪽 상단의 "+ 새 프로젝트 만들기" 버튼을 클릭합니다
  3. 프로젝트 이름을 입력합니다 (예: "Gemini Project")
  4. "프로젝트 만들기" 버튼을 클릭합니다

프로젝트가 생성되면 위와 같은 화면이 표시됩니다

3 무료 체험판 등록하기 (선택)

무료 체험판에 등록하면 $300 (약 43만원)의 무료 크레딧을 받을 수 있습니다.

안심하세요! 무료 체험 중에는 자동 결제가 진행되지 않습니다. 일반 계정을 직접 활성화하거나 선불 결제를 선택하는 경우에만 요금이 청구됩니다.

3-1. 결제 정보 확인 (1단계)

Google Cloud Platform 서비스 약관에 동의합니다.

서비스 약관 동의 화면

3-2. 고객 정보 입력 (2단계)

계좌 유형, 세금 정보, 주소를 입력합니다.

  • 계좌 유형: "개인" 선택
  • 세금 정보: "사업자 유형: 개인" 확인
  • 주소: 본인 주소 입력

고객 정보 입력 화면

3-3. 결제 수단 등록

카드 번호를 입력하고 "무료로 시작하기" 버튼을 클릭합니다.

카드 정보 입력 후 "무료로 시작하기" 클릭

주의: 카드 정보는 본인 확인 용도로만 사용됩니다. 무료 체험 기간 동안 자동으로 요금이 청구되지 않으니 안심하세요.

4 무료 크레딧 확인하기

등록이 완료되면 $300 (약 43만원)의 무료 크레딧이 지급됩니다.

Free Trial 크레딧 확인 - 90일간 사용 가능

  • 크레딧 금액: $300 (약 ₩430,298)
  • 사용 기간: 90일
  • 자동 청구: 없음 (직접 활성화 전까지)

5 API 키 발급받기

이제 감린이에서 사용할 API 키를 발급받습니다.

  1. 왼쪽 메뉴에서 "API 키" 또는 "Get API key"를 클릭합니다
  2. 오른쪽 상단의 "API 키 만들기" 버튼을 클릭합니다
  3. 프로젝트를 선택합니다 (앞서 생성한 프로젝트)
  4. 생성된 API 키를 복사합니다

API 키가 생성된 화면 - 키 옆의 복사 버튼을 클릭하세요

중요! API 키는 비밀번호처럼 안전하게 보관하세요. 타인에게 공유하거나 공개된 곳에 올리지 마세요.

6 감린이에 API 키 등록하기

복사한 API 키를 감린이에 등록합니다.

  1. 감린이 앱에서 설정 메뉴로 이동합니다
  2. "AI 채점 API 키" 입력란을 찾습니다
  3. 복사한 API 키를 붙여넣기 합니다
  4. "저장" 버튼을 클릭합니다
완료! 이제 감린이에서 AI 채점 기능을 사용할 수 있습니다.

자주 묻는 질문

Q. API 키를 잃어버렸어요

Google AI Studio의 "API 키" 메뉴에서 기존 키를 확인하거나 새 키를 발급받을 수 있습니다.

Q. 무료 한도를 초과하면 어떻게 되나요?

일일 무료 한도(RPD)를 초과하면 당일은 AI 채점을 사용할 수 없습니다. 다음 날 자정(UTC 기준)에 한도가 초기화됩니다.

Q. 결제 정보를 등록하지 않아도 되나요?

결제 정보 없이도 API를 사용할 수 있지만, 일일 약 20회(RPD)로 매우 제한적입니다. 감린이를 원활하게 사용하려면 무료 체험판 등록을 권장합니다. $300 무료 크레딧으로 충분히 사용할 수 있습니다.

Q. API 키가 작동하지 않아요

API 키를 복사할 때 앞뒤 공백이 포함되지 않았는지 확인하세요. 문제가 지속되면 새 API 키를 발급받아 다시 시도해보세요.
